## Getting Started
To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/ataisi-nathan/CloudKnightOps.git
2. Install dependencies:
    npm install
3. Start the development server:
    npm run dev
4. Open your browser and visit http://localhost:5137 to view the site.
